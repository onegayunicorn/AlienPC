# AlienPC Core Attributes

| Attribute                     | Value |
|------------------------------|-------|
| **Base Distribution**        | Debian Linux |
| **Portable Execution**       | USB / SD / E‑SATA (no install) |
| **Architecture**             | 64‑bit (x86_64) |
| **RAM Requirement**          | 1 GB – 4 GB |
| **Open‑Source Apps**         | All pre‑installed tools are GPL/LGPL |
| **Clonable Images**          | `dd`‑compatible `.img` files |
| **Legacy Revival**           | Works on PCs dating back to 2005 with minimal drivers |
| **Custom Boot Envs**          | GRUB, Syslinux, UEFI fallback |
| **Footprint**                | ~300 MB compressed rootfs |

## Philosophy
AlienPC is built on the philosophy of portability and freedom, aiming to revitalize legacy hardware with a minimal footprint while providing advanced genomic analysis capabilities.
